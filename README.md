# CalmOps Website

Simple, clean landing page for CalmOps digital products.

## Stack
- Pure HTML + CSS (no frameworks)
- Mobile responsive
- Fast loading

## Deployment

### Vercel
```bash
# Install Vercel CLI (if needed)
npm i -g vercel

# Deploy
vercel
```

### Netlify
```bash
# Install Netlify CLI (if needed)
npm i -g netlify-cli

# Deploy
netlify deploy --prod
```

Or use Netlify drag-and-drop deploy: https://app.netlify.com/drop

## Pages
- `index.html` - Main landing page
- `terms.html` - Terms of Service
- `refund-policy.html` - Refund Policy
- `style.css` - Styles for all pages
